# Context API
- The React Context API is a built‑in state management solution that allows you to share data across the component tree without passing props manually at every level (avoiding “prop drilling”).
    ## prop drilling
    - Prop drilling in React refers to passing data through multiple nested components just to reach a deeply nested child, even when intermediate components don’t need that data. This often makes the code harder to maintain and understand.

     ### when to use
     - Many components need the same data.
     - Passing props through multiple nested components becomes messy.
